# 第10章：経営指標の相互関係（指標のつながりを理解する）【超詳細版】

## 1. 指標の相互関係とは
経営分析の指標は単独で見るのではなく、「つながり」で理解することが重要。

例えば：
- 安全性が低いと収益性にも悪影響が出る
- 効率性が低いと資金繰りが悪化し安全性が下がる
- 成長性が高いと投資CFがマイナスになりやすい

このように、指標は互いに影響し合っている。

---

# 2. 安全性と収益性の関係

## ■ 自己資本比率が低い企業
→ 借入依存が高い  
→ 金利負担が増える  
→ 経常利益率が低下する  
→ ROEは一時的に高く見えることもある（レバレッジ効果）

ポイント：
- 安全性が低い企業は収益性も不安定になりやすい

---

## ■ 自己資本比率が高い企業
→ 借入が少ない  
→ 金利負担が小さい  
→ 経常利益率が安定  
→ 不況に強い

ポイント：
- 安全性が高い企業は収益性も安定しやすい

---

# 3. 効率性と収益性の関係

## ■ 総資本回転率が低い企業
→ 資産を使いすぎている  
→ 売上が伸びにくい  
→ 営業利益率が低下する

例：
- 在庫が多すぎる
- 設備投資が過剰

---

## ■ 総資本回転率が高い企業
→ 少ない資産で売上を生む  
→ 営業利益率が高くなりやすい  
→ ROAも高くなる

ポイント：
- 効率性は収益性に直結する

---

# 4. 成長性と安全性の関係

## ■ 成長性が高い企業
→ 設備投資が増える  
→ 投資CFが大きくマイナス  
→ 借入が増える  
→ 財務CFがプラスになりやすい  
→ 安全性が低下することもある

ポイント：
- 成長企業は安全性が低く見えることがある（投資のため）

---

## ■ 成長性が低い企業
→ 投資が少ない  
→ 投資CFが小さい  
→ 借入も少ない  
→ 安全性は高いが収益性が伸びない

ポイント：
- 成長性と安全性はトレードオフの関係になりやすい

---

# 5. キャッシュフローと安全性・収益性の関係

## ■ 営業CFがプラス
→ 本業が強い  
→ 安全性が高い  
→ 収益性も安定しやすい

---

## ■ 営業CFがマイナス
→ 本業が現金を生んでいない  
→ 借入で資金繰りする必要がある  
→ 財務CFがプラスになりやすい  
→ 安全性が低下する  
→ 収益性も悪化しやすい

---

## ■ 投資CFがマイナス
→ 成長のための投資  
→ 長期的には収益性向上につながる

---

## ■ 投資CFがプラス
→ 資産売却で現金化  
→ 再建中の企業に多い  
→ 安全性は改善するが収益性は低いことが多い

---

# 6. 指標のつながりを理解するための典型パターン

## ■ 優良企業のパターン
- 営業CF：プラス
- 投資CF：マイナス（成長投資）
- 財務CF：マイナス（借入返済）
- 安全性：高い
- 収益性：高い
- 効率性：高い
- 成長性：高い

---

## ■ 再建中企業のパターン
- 営業CF：プラス
- 投資CF：プラス（資産売却）
- 財務CF：プラス（借入増加）
- 安全性：低い
- 収益性：低い
- 効率性：低い

---

## ■ 危険企業のパターン
- 営業CF：マイナス
- 投資CF：プラス（資産売却）
- 財務CF：プラス（借入依存）
- 安全性：低い
- 収益性：低い
- 効率性：低い
- 成長性：低い

---

# 7. 試験での重要ポイントまとめ
- 指標は単独でなく「つながり」で理解する
- 安全性が低いと収益性も悪化しやすい
- 効率性が低いと資金繰りが悪化し安全性が下がる
- 成長性が高い企業は投資CFがマイナスになりやすい
- 営業CFが最重要の指標である
